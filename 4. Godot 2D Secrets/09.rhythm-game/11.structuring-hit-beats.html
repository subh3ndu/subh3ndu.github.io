The HitBeat stack

Thanks to the Synchronizer, we already have a marker for each half-beat as a track plays.

To determine whether we instance a HitBeat on any given half-beat, weâ€™ll have a stack (array) of instructions stored in the HitSpawner.

Every time the Events.beat_incremented signal fires, the HitSpawner will grab the next instruction and either instance a HitBeat or do nothing.

Weâ€™ll also use scenes to visually design patterns and create unique stacks for each track. But first, letâ€™s hard-code a simple pattern so you can see how this stack approach works.

# Here, we populate the stack.
# Every even half-beat, we add a color and position information.
# On the odd half-beats, we add an empty dictionary to the stack.
func _generate_stack() -> void:
    # We'll generate 5 buttons to tap aligned diagonally.
    for i in range(10):
        # An empty dictionary means "spawn nothing."
        var hit_beat_data = {}

        # Every even half-beat, we calcualte a new color and position.
        if i % 2 == 1:
            hit_beat_data.color = int(rand_range(0, 5))
            hit_beat_data.global_position = i * Vector2(100, 100)

        # And we append every dictionary to the stack.
        _stack_current.append(hit_beat_data)

Now we have a stack, weâ€™re going to rewrite _spawn_beat() to pop dictionaries from the _stack_current and process them. Replace the existing method with the following.

func _spawn_beat(msg: Dictionary) -> void:

    if not enabled:
        return

    # After we've processed everything in the stack, we disable the HitSpawner
    # We also emit a signal which we can use later to show an end screen
    if _stack_current.empty():
        enabled = false
        Events.emit_signal("track_finished")
        return

    # pop_front() grabs the front dictionary from the stack and also removes it
    var hit_beat_data: Dictionary = _stack_current.pop_front()

    # If the dictionary has no global_position, we skip instancing
    if not hit_beat_data.has("global_position"):
        return

    # Otherwise we add extra information as we did before and instance a HitBeat
    hit_beat_data.bps = msg.bps
    hit_beat_data.half_beat = msg.half_beat

    var new_beat: Node = hit_beat.instance()
    add_child(new_beat)
    new_beat.setup(hit_beat_data)
